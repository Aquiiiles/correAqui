# CorreAqui

App mobile de descoberta de eventos esportivos. Expo + TypeScript + React Native.

## Comandos

- `npx expo start` — inicia dev server
- `npx tsc --noEmit` — verifica tipos

## Estrutura

- `src/constants/tokens.ts` — design tokens (cores, fontes, espaçamentos)
- `src/constants/data.ts` — dados mock
- `src/components/` — componentes reutilizáveis
- `src/screens/` — telas do app
- `src/types/` — tipos TypeScript

## Convenções

- Branch principal: `master`
- Idioma do app: pt-BR
- Tema de cor: Pulso (verde, accent #2e8b57)
- Fontes: Hanken Grotesk (UI) + Space Grotesk (dados técnicos)

# CorreAqui

Todo esporte da sua cidade, num lugar só.

App mobile (iOS/Android) para descobrir e criar eventos esportivos por localidade e data.

## Stack

- React Native + Expo + TypeScript
- Fontes: Hanken Grotesk + Space Grotesk
- Tema: Pulso (verde)

## Desenvolvimento

```bash
npm install
npx expo start
```
